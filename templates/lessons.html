Об уроках

Каждый урок содержит теоретические материалы, видео-контент и практические задания. Уроки структурированы по курсам и позволяют поэтапно изучать материал.

API Endpoints для уроков

Полный набор API для работы с уроками через RESTful интерфейс.

GET /api/courses/lessons/ - Получить список всех уроков
Возвращает пагинированный список уроков с информацией о названии, описании и курсе
POST /api/courses/lessons/ - Создать новый урок
Создает новый урок (требует авторизации и соответствующих прав)
GET /api/courses/lessons/{id}/ - Получить детали урока
Возвращает полную информацию об уроке, включая ссылку на видео
PUT /api/courses/lessons/{id}/ - Обновить урок
Обновляет информацию об уроке (требует авторизации и соответствующих прав)
DELETE /api/courses/lessons/{id}/ - Удалить урок
Удаляет урок (требует авторизации и соответствующих прав)

Структура данных урока

Модель Lesson:
  • title - Название урока (CharField, max_length=100)
  • preview - Превью урока (ImageField)
  • description - Описание урока (TextField)
  • video_link - Ссылка на видео (URLField)
  • course - Связь с курсом (ForeignKey)
