API для курсов

Используйте наш REST API для получения информации о курсах, создания новых курсов и управления существующими.

Доступные эндпоинты:
  • GET /api/courses/ - список курсов
  • POST /api/courses/ - создать курс
  • GET /api/courses/{id}/ - детали курса
  • PUT /api/courses/{id}/ - обновить курс
Особенности:
  • JWT аутентификация
  • Фильтрация и поиск
  • Пагинация
  • Права доступа
