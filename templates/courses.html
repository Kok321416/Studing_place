О курсах

Наши курсы разработаны опытными преподавателями и содержат актуальную информацию по современным технологиям. Каждый курс включает в себя теоретические материалы, практические задания и видео-уроки.

API Endpoints для курсов

Полный набор API для работы с курсами через RESTful интерфейс.

GET /api/courses/ - Получить список всех курсов
Возвращает пагинированный список курсов с информацией о названии, описании и количестве уроков
POST /api/courses/ - Создать новый курс
Создает новый курс (требует авторизации и соответствующих прав)
GET /api/courses/{id}/ - Получить детали курса
Возвращает полную информацию о курсе, включая список уроков
PUT /api/courses/{id}/ - Обновить курс
Обновляет информацию о курсе (требует авторизации и соответствующих прав)
DELETE /api/courses/{id}/ - Удалить курс
Удаляет курс (требует авторизации и соответствующих прав)

Структура данных курса

Модель Course:
  • title - Название курса (CharField, max_length=100)
  • preview - Превью курса (ImageField)
  • description - Описание курса (TextField)
  • lessons_count - Количество уроков (вычисляемое поле)
  • lessons - Список уроков курса (связанные объекты)

Поиск курсов

Используйте API для поиска курсов по различным критериям

GET /api/courses/?search=название

Фильтрация

Фильтруйте курсы по различным параметрам

GET /api/courses/?ordering=title

Пагинация

Получайте курсы постранично для оптимизации

GET /api/courses/?page=1
