API для уроков

Используйте наш REST API для работы с уроками: просмотр, создание, редактирование и удаление.

Доступные эндпоинты:
  • GET /api/lessons/ - список уроков
  • POST /api/lessons/ - создать урок
  • GET /api/lessons/{id}/ - детали урока
  • PUT /api/lessons/{id}/ - обновить урок
Особенности:
  • Связь с курсами
  • Медиа файлы
  • Права доступа
  • Фильтрация
