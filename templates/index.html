API Endpoints

Наша платформа предоставляет RESTful API для интеграции с внешними системами и создания мобильных приложений.

GET /api/courses/ - Список всех курсов
POST /api/courses/ - Создание нового курса
GET /api/courses/{id}/ - Детали курса
PUT /api/courses/{id}/ - Обновление курса
DELETE /api/courses/{id}/ - Удаление курса
GET /api/courses/lessons/ - Список всех уроков
GET /api/users/ - Список пользователей (требует авторизации)
